# just-deploy
Deploy projects without any extra setup.

## 1. Install

```
composer require dfba/just-deploy:0.1.*
```

or:

```
php composer.phar require dfba/just-deploy:0.1.*
```

## 2. Configure

## 3. Deploy

Windows:

```
vendor\bin\just-deploy
```

Linux/Unix:

```
./vendor/bin/just-deploy
```
